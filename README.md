# 🏏 IPL Data Analysis & Dashboard

An end-to-end data analytics project on IPL (2008–2020) including data cleaning, visualization, SQL analysis, machine learning, and an interactive dashboard.

---

## 👨‍💻 Contributors

* Hashwin — EDA, Charts, Machine Learning, Dashboard
* Hema — Data Cleaning, Feature Engineering, SQL

---

## 🔧 Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn, Plotly
* Scikit-learn
* SQLite
* Streamlit

---

## 📊 Features

* 📈 Data Cleaning & Preprocessing
* 📊 Exploratory Data Analysis
* 📉 5+ Visualizations
* 🧠 Machine Learning Model (Random Forest + Logistic Regression)
* 🌐 Interactive Dashboard (Streamlit UI)

---

## 🚀 Run Project

### Install dependencies:

pip install -r requirements.txt

### Run notebook:

jupyter notebook IPL_Analysis.ipynb

### Run dashboard:

streamlit run app.py
